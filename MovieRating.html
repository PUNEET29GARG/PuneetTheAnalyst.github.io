Movie Ratings Analysis
  • Projects

This project demonstrates how to: :

  • Import libraries such as numpy, pandas, matplotlib, and seaborn.

  • Read data files containing movie ratings, movie information, and user information.

  • Clean and manipulate data such as changing column names, data types, and extracting year from date.

  • Plot data using various charts such as count plots, histograms, and scatter plots.

  • Group and aggregate data using the groupby and agg methods.

  • Join data frames using the join method.

  • Filter and sort data using the indexing operator and the sort_values method.
